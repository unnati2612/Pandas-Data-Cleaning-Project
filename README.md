# Pandas Data Cleaning Project

## Objective
Learn Python basics and perform data exploration and cleaning using Pandas.

## Tasks Performed
- Loaded CSV dataset into a Pandas DataFrame
- Explored data using:
  - head()
  - tail()
  - shape
  - columns
  - dtypes
- Handled missing values
- Performed filtering and column selection
- Removed duplicate rows
- Created derived column: total_amount
- Saved cleaned dataset as a new CSV file

- Python

- Jupyter Notebook
- VS Code
- Kaggle
